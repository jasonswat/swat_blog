blog_pages
---------

A [Jekyll](https://jekyllrb.com)-based blog for netlify.

## Local development

Requirements:
- Ruby 2.0+
- Jekyll and Bundler gems

```sh
gem install jekyll bundler
bundle install
bundle exec jekyll serve
```
